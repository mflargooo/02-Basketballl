# 02 Basketballl

README

        "Bunny Ball" developed by Hidden Reflection Studios, puts a spin on basketball

RUN DIRECTIONS

        Windows: Unzip the file and run the .exe directly in the uzipped folder.
            Mac: Run the .app file as is.
